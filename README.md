## Tecnologias

- NestJS
- Prisma
- PostgreSQL
- JWT
- Bcrypt
- Zod
- React
- NextJS
- TailwindCSS

## Estrutura do Projeto

- apps/api: Contém o backend do aplicativo.
- apps/web: Contém o frontend do aplicativo.


## Iniciar o projeto

1. Instalar as dependências

```bash
cd apps/api
npm install
npx prisma db push
```

```bash
cd apps/web
npm install
```

2. Rodar o projeto

<!-- backend -->

```bash
cd apps/api
npm run start:dev
```

<!-- frontend -->

```bash
cd apps/web
npm run dev
```
